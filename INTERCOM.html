Alert: Be careful while accessing intercom

Namma Falguni Intercom service allows entry to a visitor to a building. A visitor can come up to the entry system at the main building entrance, select an entry and call a resident. This sends a buzzer signal to the resident inside the apartment.

How does intercom work?
Namma Falguni intercom system works by sending a notification from one device to another, which can be picked up as an audio or video call. Currently, wired intercom system available. Wireless intercom is being set up at Namma Falguni. Click on the progress button to know how much work has been completed.
How to use?
The receiver at your homes can be accessed. It will connect you with the visitor and communication can be made. You can allow them by clicking on green button.
